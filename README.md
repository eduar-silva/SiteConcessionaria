Site Concessionaria 
</h1>
<p>Site para concessionária fícticia apenas para estudo.</p>

Site para uma concessionária com cadastro de funcionários e clientes. As linguagens sendo usadas são HTML, PHP, CSS, JavaScript. O site também contém um banco de dados (MySQL).

# Tecnologias Usadas
``Bootstrap`` ``MySQL`` ``JavaScript`` ``PHP``

# Uso e Funcionamento

<p>Site onde o usuário registra nome dos funcionários, clientes, marcas, carros e vendas. Após o registro de cada categoria é possível a visualização, remoção e edição dos dados registrados. Nas categorias carros e vendas foram usados INNER JOIN. Na categoria de carros o INNER JOIN foi usado para registrar a marca do carro, e nas vendas foi usado para registrar o carro vendido.</p>
